\doxysubsection{Detailed Description}
A class to provide time-\/related functionalities. 

The \doxylink{class_clock}{Clock} class provides various time-\/related functionalities such as measuring elapsed time, retrieving current time in different formats, and providing utilities to get the day of the week, month, year, etc. It supports different time units for measuring elapsed time and provides thread-\/safe timer operations.
