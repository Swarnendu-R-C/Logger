Implementation of the \doxylink{class_clock}{Clock} class for time measurement and formatting. 

This file contains the implementation of the \doxylink{class_clock}{Clock} class, which provides functionalities for measuring elapsed time, getting the current time in various formats, and retrieving specific components of the current time.
