\doxysubsection{Detailed Description}
Helper utilities and wrapper functions for the \doxylink{classlogger_1_1_logger}{Logger} library. 

This header provides a set of template utilities and convenience functions to simplify logging operations using the \doxylink{classlogger_1_1_logger}{Logger} class. It includes type traits for container detection, functions for setting logger properties, and various logging wrappers for different log levels (info, warning, error, debug, fatal, etc.), as well as support for logging lists and assertion failures.
